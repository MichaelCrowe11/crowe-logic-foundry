Get running in four steps

  1. Step 1 · Install
    git clone https://github.com/MichaelCrowe11/crowe-logic-foundry.git && cd crowe-logic-foundry && pip install -e .
  2. Step 2 · Set your API key
    echo 'CROWE_LOGIC_API_KEY=YOUR_KEY' >> ~/.config/crowe-logic/.env
  3. Step 3 · Launch
    crowe-logic
  4. Step 4 · Verify your balance
